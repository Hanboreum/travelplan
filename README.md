# All_In_Won
사용자 인터페이스 및 실습 수업 프로젝트입니다.  
자동으로 실시간 환율로 환전된 금액을 보여주는 해외여행 가계부입니다.

## 사용한 외부 라이브러리/API
* Apache POI: 가계부 내역을 엑셀파일로 출력하는데 사용함
* MPAndroidChart: 통계 화면에 그래프를 그리는데 사용함
* Exchange rates API (https://exchangeratesapi.io/): 실시간 환율을 가져오는데 사용함
